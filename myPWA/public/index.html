    • About

  • This project is designed to create a website holding a database for games that allows you to search for specific games in a variety of ways, such as by name, and developer whilst also putting constraints like genre, release date and compatibility with multiplayer. This project will use github codespaces and repositories to carry out the project, with the website running universally on every website. As for hardware, my personal laptop will be used for coding and my home computer and phone will be used for testing the website in a variety of circumstances. The success of this project will be measured by: how well the user can find games within the website using both narrow and wide searches, enable the discovery of new games, manage accessibility options with website clarity and allow the user to seamlessly use the website with both palatable visuals and understandable UI.
